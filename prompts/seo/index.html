10 ChatGPT Prompts for SEO That Cut Your Workflow Time in Half

SEO is increasingly about content quality and topical authority. These prompts help you produce the right content faster — from keyword research to meta tags to schema markup.

Prompt 1 of 5
Keyword cluster generator
Generate a keyword cluster for the seed keyword "[SEED KEYWORD]". Organize into: (1) 5 head terms (high volume, high competition), (2) 10 body keywords (medium volume, medium competition), (3) 15 long-tail keywords (lower volume, easier to rank). For each keyword, estimate: search intent (informational/navigational/commercial/transactional) and a one-word difficulty estimate (Easy/Medium/Hard). Format as a table.
💡 Pro tip: Target long-tail keywords first. Once you rank for 10+ long-tail variations, your topical authority makes head terms easier.
Prompt 2 of 5
Content brief (SEO-optimized)
Create a detailed SEO content brief for the keyword "[TARGET KEYWORD]" (search intent: [INTENT]). Include: recommended title options (3), H2/H3 structure, word count target, required entities to mention, LSI keywords to include (10), competing pages to analyze, internal linking opportunities, and schema markup type recommendation. Audience: [AUDIENCE].
💡 Pro tip: Giving the brief to a writer (or ChatGPT for a first draft) cuts editing time by 50%.
Prompt 3 of 5
Meta title and description optimizer
Optimize the meta title and description for this page: Title: "[CURRENT TITLE]", Page topic: "[TOPIC]", Target keyword: "[KEYWORD]". Write 3 versions of each (title under 60 chars, description under 155 chars). Prioritize: primary keyword in title, benefit/hook in description, natural language. Flag if any version would likely trigger truncation.
💡 Pro tip: Test all 3 versions using Google's SERP snippet preview tool before choosing.
Prompt 4 of 5
FAQ schema generator
Generate 5 FAQ questions and answers for a page about "[TOPIC]". The questions should: match real search queries people ask about [TOPIC], be answerable in 50-75 words each, include the target keyword "[KEYWORD]" naturally in at least 2 answers, and provide genuine value. Format as valid JSON-LD FAQ schema ready to paste into <script type="application/ld+json">.
💡 Pro tip: FAQ schema can capture featured snippets and 'People Also Ask' boxes without ranking changes.
Prompt 5 of 5
Internal linking strategy
Create an internal linking strategy for a website in [NICHE] with these core pages: [LIST YOUR KEY PAGES]. For each page, suggest: 3 pages that should link TO it (with anchor text suggestions), and 3 pages it should link OUT to. Prioritize by: topical relevance, conversion value, and current ranking potential.
💡 Pro tip: Internal links pass authority from strong pages to weaker ones. Map this quarterly.
